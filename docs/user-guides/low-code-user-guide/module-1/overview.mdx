# Module 1: Quick Start

## Quick Start Overview
In this module, you will be given exercises to learn the basics and begin using EASYProcess quickly. By the end of this you will be able to:

- Navigate EASYProcess Design Time, RunTime, and App Suite
- Use the IDE for development and test on RunTime desktop or Mobile App
- Understand the Model-View-Controller (MVC) architecture of Views in EASYProcess
- Create a WebPage
- Populate the Model of a View
- Dynamically populate values in Controls on a WebPage
- Use a Controller to update the Model Data and make a change on the page
- Use User Session Variables
- Configure database services
- Query a database using an Inquire service
- Learn XSLT uses, formatting, and use XSLT functions
- Use timezone converting date services
- Learn the difference between an EASYProcess View List and Grid
- Manually create a List with a paging and filtering controller
- Use the List/Grid Wizard to create a List
- Learn about Submethods vs Stand-Alone Methods and how to use both
- Use a Loop, an XSLT for-each, and a ToDate service to iterate on an array and modify data
